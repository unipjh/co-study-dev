> 2026.04.06 기준 구현 현황 정리

---

## 앱의 전체 흐름

```
라이브러리 (홈화면)
   └── PDF 업로드 / 목록 확인
         └── 뷰어 (문서 읽기 화면)
               ├── 텍스트 선택 → 하이라이트 + 메모 저장
               ├── 사이드바 > Chat 탭 → AI와 대화
               └── 사이드바 > 메모 탭 → 내 하이라이트 모아보기
```

---

## 화면 1 · 라이브러리

> 앱을 켜면 처음 보이는 화면

- PDF 파일을 업로드할 수 있어요
- 이전에 올린 문서들이 카드 형태로 쭉 나와요
- 카드 클릭 → 해당 PDF 뷰어로 이동
- 카드 우상단 × 버튼 → 문서 삭제

---

## 화면 2 · 뷰어

> PDF를 실제로 읽는 화면. 크게 왼쪽(PDF)과 오른쪽(사이드바)으로 나뉘어요.

### 상단 툴바

| 기능 | 설명 |
|------|------|
| 뷰 모드 토글 | **페이지** (한 장씩) / **스크롤** (전체 연속 스크롤) 전환 |
| 줌 | − / % / + 로 0.5배 ~ 2배 조절 |
| 페이지 이동 | 페이지 모드일 때 ← / → 로 이동 |
| 사이드바 토글 | 오른쪽 패널 열기/닫기 |

---

## 기능 1 · 하이라이트 & 메모

> PDF에서 텍스트를 드래그하면 시작돼요

```
텍스트 드래그
   └── 선택 툴바 등장
         ├── 🟡 노란색 → "중요"
         ├── 🔵 파란색 → "이해 필요"
         └── 🟢 초록색 → "암기"
               └── (선택 후) 메모 입력 가능
                     └── 저장 → PDF 위에 형광펜 표시
```

- 같은 자리를 다시 드래그하면 기존 하이라이트는 자동으로 덮어써져요
- 하이라이트를 클릭하면 팝업이 떠서 메모 수정 / 색상 변경 / 삭제 / Chat으로 보내기 가 가능해요

---

## 기능 2 · Chat (AI 대화)

> 사이드바 첫 번째 탭

- 텍스트를 그냥 입력해서 AI에게 질문할 수 있어요
- **하이라이트 팝업 → "Chat으로 보내기"** 를 누르면, 그 텍스트가 문맥으로 채팅창에 딸려 와요
- 빠른 버튼 2개:
  - **설명** → 선택한 내용을 AI가 쉽게 설명해줘요
  - **퀴즈 생성** → 내용 기반 퀴즈를 만들어줘요
- AI 답변은 실시간으로 타이핑되듯 스트리밍돼요
- 대화 내용은 자동 저장돼요

---

## 기능 3 · 메모 탭

> 사이드바 두 번째 탭

- 내가 만든 하이라이트를 **페이지 순서대로** 모아서 보여줘요
- 각 카드에는 색상 태그, 페이지 번호, 원문 텍스트, 메모 내용이 표시돼요
- 카드 클릭 → 해당 페이지로 이동
- "Chat으로 보내기" 버튼 → 그 하이라이트로 AI 대화 시작

---

## 데이터 저장 구조

> 모든 데이터는 Firebase에 자동 저장·동기화돼요 (새로고침해도 사라지지 않아요)

```
Firebase
├── 문서 목록 (PDF 이름, 업로드 날짜 등)
├── 하이라이트 & 메모 (문서별로 분리 저장)
└── 채팅 내역 (문서별로 분리 저장)
```
